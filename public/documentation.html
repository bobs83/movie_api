Achievement 2 - Project: myFlix

Goal: To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

API Endpoints

Request URL HTTP Method Request body data format Response body data format
Get a list of ALL movies /movies GET none A JSON object holding an array with all movies.
Get data about a single movie by title /movies/[title] GET none A JSON object holding data about a single movie, containing description, genre, director, image URL, whether it’s featured or not. Example:
{"title": "Split", "description": "The film follows a man with dissociative identity disorder who kidnaps...", "genre": { "name": "thriller", "description": "This film is a ..." }, "director": { "name": "M. Night Shyamalan", }, "imageURL": "https://www.google.com...", "featured": false}
Get data about genre /movies/genre/:genreName GET none A JSON object holding data about a genre containing name and description properties. Example:
"Genre": { "Name": "Triller", "Description": "Is a genre of fiction that ..." }
Get data about a director by name /movies/directors/:directorName GET none A JSON object holding an array with directors name.
Add a new user /users POST A JSON object holding user data with structure:
{ name: "Bobby", favoriteMovies: [] } 		A JSON object holding user data plus an ID:
{ id: 1, name: "Bobby", favoriteMovies: [] }
Update user username /users/:id PUT A JSON object holding user data with structure: { name: "Sven", favoriteMovies: [] } A JSON object holding user data plus an ID: { id: 1, name: "Sven", favoriteMovies: [] }
Add a movie to user favorite list /users/:id/:movietitle POST None A text message indicating whether a movie was successfully added to the favorite list.
Remove a movie from user favorite list /users/:id/:movieTitle DELETE None A text message indicating whether a movie was successfully removed from the favorite list.
Delete user registration /users/:id DELETE None A text message indicating whether a user was successfully removed.
